TubeTrend AI integrates with the following external services:

  • YouTube Data API v3: To fetch video statistics and trending content (Google's Privacy Policy applies)

When you use these services, your API requests are sent directly from your device to their servers. We do not intercept or store this communication.

4. Data Security

We implement industry-standard security measures:

  • Encrypted local storage for API keys
  • No data transmission to external servers (except API calls you initiate)
  • Open-source codebase for transparency and auditing
  • Regular security updates and patches

You have complete control over your data:

  • Access: View all stored data in the app's Settings and Analytics screens
  • Deletion: Delete API keys and usage data anytime from Settings
